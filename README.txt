

# factorio_saves_synch
Cинхронизирует сохранения факторио через ядиск при игре с разных устройств
-----------------------------------------------------------------------------------
factorio_online.bat       - запускает оболочку факторио, которая проверяет,
                            не играет ли кто-то сейчас, потом запускает факторио,
                            а после игры сихронизирует сохранения
update_saves.bat          - просто синхронизирует сохранения
-----------------------------------------------------------------------------------
Для работы необходимо:
- установить python, если его нет
- создать яндекс аккаунт, если его нет
- создать на ядиске (https://disk.yandex.ru/client/disk) папку /factorio/
- получить отладочный OAuth-токен для своего ядиска
    - пункты 1 и 2: https://yandex.ru/dev/disk/api/concepts/quickstart.html
- в файле settings.json указать:
    - yadisk_token        - полученный токен
    - factorio_exe_path   - путь к екзешнику факторио
    - factorio_saves_path - путь к папке сохранений
    - name                - свой ник
