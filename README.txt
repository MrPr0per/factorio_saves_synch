

# factorio_saves_synch
Cинхронизирует сохранения факторио через ядиск при игре с разных устройств
-----------------------------------------------------------------------------------
Запускать launch.bat
-----------------------------------------------------------------------------------
Для работы необходимо:
- в файле settings.txt указать свой ник (name), 
                               путь к екзешнику факторио (factorio_launch_path), 
                               путь к папке сохранений (factorio_saves_path), 
                               свой ядиск токен (yadisk_token)
- создать на своем ядиске папку /factorio/
- в этой папке создать пустой файл players_online.txt
